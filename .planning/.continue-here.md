<context>

mature: spawn N agents in isolated worktrees with truly disjoint file
scopes, merge cleanly with `--no-ff`, verify with build + tests + push,

can bypass the worktree branch and push directly to feature/qwen3.6 if
their isolation mode permits it (Gemma 3 agent did so this session; the
auditor did the same last session). Tolerable as long as their work is
small + self-contained.

The audit-H3 trap-the-bug discovery (IQ3 dispatch arms wired but
unreachable due to missing upload predicates) is exactly the
class-of-bug the structural-mitigation discipline is supposed to catch.
Caught here only because the IQ3 host parity tests were written —
without them the dispatch would have stayed silently dead until a real
IQ3 GGUF hit the bench.
